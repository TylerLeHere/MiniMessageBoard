# Mini Message Board

A simple message board built with [Express.js](https://expressjs.com/) and [EJS](https://ejs.co/). Users can view messages, submit new ones, and see them instantly on the homepage.

---

## What It Does

- Displays a list of all messages on the homepage
- Lets users submit a new message through a form
- Each message shows:
  - Author's name
  - Message text
  - Date added
- Redirects back to the homepage after submitting
- Each message has a link for potential detail pages in the future
